# JavaScript & TypeScript & React [중급 트랙]

이 프로젝트는 자바스크립트 기본기를 마친 후, 타입스크립트(TypeScript) 정적 형식 검사와 리액트(React) 컴포넌트 설계 및 상태 관리 기초를 심도 있게 학습하기 위한 중급 실습 트랙입니다.

## 📅 학습 핵심 로드맵 [중급]

- **Chapter 01**: TypeScript 설치 및 컴파일러 환경 구성 (`tsconfig.json` 분석)
- **Chapter 02**: TypeScript 기본 타입 바인딩 (원시값, 객체, 배열, Union/Intersection)
- **Chapter 03**: 인터페이스와 타입 별칭의 차이 (`interface` vs `type`)
- **Chapter 04**: TypeScript 함수 타이핑 및 선택적 매개변수
- **Chapter 05**: TypeScript 제네릭(Generic) 기초 (타입 매개변수의 안전성 확보)
- **Chapter 06**: TypeScript 고급 유틸리티 타입 (`Omit`, `Pick`, `Partial`, `Readonly`)
- **Chapter 07**: React 가상 DOM의 실체와 빌드 시스템 이해 (Vite, JSX 컴파일)
- **Chapter 08**: React 컴포넌트 Props 및 State 타이핑과 기본 컴포넌트 설계
- **Chapter 09**: `useState`를 이용한 반응형 상태 제어와 불변성 업데이트 법칙
- **Chapter 10**: `useEffect` 생명주기와 부수 효과 제어 (Cleanup 함수의 중요성)
- **Chapter 11**: React 커스텀 훅(Custom Hooks) 설계 및 상태 로직 재사용
- **Chapter 12**: React Context API를 통한 상태 전파 및 리렌더링 최적화 한계
- **Chapter 13**: Zustand 라이브러리를 이용한 Flux 아키텍처 전역 상태 관리
- **Chapter 14**: Jest와 React Testing Library를 이용한 컴포넌트 단위 테스트
- **Chapter 15**: [중급 종합 프로젝트] TypeScript + React + Zustand 기반 영화 검색 대시보드 구축
