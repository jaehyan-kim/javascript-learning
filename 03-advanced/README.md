# Frontend Architecture & Performance [고급 트랙]

이 프로젝트는 자바스크립트와 리액트의 기본기, 중급 과정을 마친 후, 가상 DOM의 렌더링 원리, 상태 관리 엔진 직접 설계, 서버 사이드 렌더링(SSR), 네트워크 비동기 최적화 및 배포 파이프라인 등 실무 고급 아키텍처를 학습하기 위한 고급 트랙입니다.

## 📅 학습 핵심 로드맵 [고급]

- **Chapter 01**: 가상 DOM 렌더링 조정(Reconciliation) 및 Diffing 알고리즘 설계 원리
- **Chapter 02**: 가상 DOM 렌더러 직접 빌드 (Fiber 아키텍처 맛보기)
- **Chapter 03**: 커스텀 상태 엔진 (Redux / Zustand internals) 직접 구현
- **Chapter 04**: React 성능 최적화: `useMemo`, `useCallback`, `React.memo` 튜닝
- **Chapter 05**: 대용량 리스트 렌더링 최적화: 가상 스크롤(Virtual List) 구현
- **Chapter 06**: 동적 스레드 병렬 처리: Web Workers를 이용한 CPU 무거운 연산 분산
- **Chapter 07**: Next.js App Router 아키텍처 분석 및 Server/Client Component 경계 설계
- **Chapter 08**: React Server Components (RSC) 렌더링 동작 원리 및 Payload 분석
- **Chapter 09**: SSR (Server-Side Rendering) 및 Static Generation (SSG) 빌드 메커니즘
- **Chapter 10**: Hydration 동작 원리와 최적화 및 런타임 Hydration Mismatch 디버깅
- **Chapter 11**: 네트워크 스트리밍: ReadableStream 수집 및 실시간 chunk 파싱 가공
- **Chapter 12**: AI LLM API 연동 및 스트리밍 답변 화면 실시간 렌더링
- **Chapter 13**: 컨테이너화 가상화: Docker 파일 작성 및 로컬 개발용 컨테이너 빌드
- **Chapter 14**: 배포 자동화: GitHub Actions + Docker Registry CI/CD 파이프라인
- **Chapter 15**: [고급 종합 프로젝트] 가상 렌더링 + 커스텀 상태 스토어 + AI API 스트리밍 수신 탑재 대형 협업 대시보드 Vercel/Docker 최종 배포
