# Chapter 15: 클로저와 상태 은닉 모듈 (Closures and State Encapsulation)

## 📖 핵심 개념
자바스크립트의 정적 렉시컬 스코프와 클로저의 은닉화 특성을 깊이 이해하고, 상태 및 구독 모델(Pub/Sub)을 캡슐화하는 클로저 저장소 모듈을 직접 구현합니다.

*   **렉시컬 스코프와 클로저**:
    *   렉시컬 스코프는 코드가 선언된 정적 환경에 따라 상위 스코프가 결정되는 규칙이며, 클로저는 함수가 이 정적 환경을 기억하여 외부에서 실행될 때도 해당 변수를 계속 참조할 수 있는 특성입니다. (예: [용어 사전](../../../../GLOSSARY.md#lexical-scope) 및 [클로저](../../../../GLOSSARY.md#closure) 참고)
    *   **Friendly_Tutor의 친절한 비유**: "클로저는 은행의 비밀 금고(은닉 상태)에 접근할 수 있는 열쇠를 쥔 유일한 은행 관리인(반환된 내부 함수)과 같습니다. 금고는 바깥세상(전역 스코프)에서 완전히 감춰져 있어서 관리인을 통해서만 안전하게 예금(상태)을 넣고 뺄 수 있습니다."
*   **구독 모델과 createStore**:
    *   createStore 함수는 상태(state)와 구독자 리스트(listeners)를 렉시컬 환경에 격리시켜 두고, getState, dispatch, subscribe 같은 전용 클로저 함수들을 반환하여 안전하게 상태를 관리 및 공유합니다.
    *   **Senior_Practitioner의 실무 팁**: "구독 모델(Pub/Sub)을 개발할 때, 컴포넌트가 마운트 해제(Unmount)될 때 반환받은 unsubscribe() 함수를 호출하여 이벤트를 명시적으로 해제해 주지 않으면, 클로저 환경 내부의 listeners 배열에 등록된 컴포넌트 참조가 계속 유지됩니다. 이로 인해 가비지 컬렉터가 메모리를 회수하지 못해 메모리 누수(Memory Leak)가 발생하고 누적되어 결국 탭이 폭발하는 메모리 크래시 현상을 겪게 됩니다. 반드시 구독 해제 라이프사이클을 완성하십시오."
*   **실행 컨텍스트 환경과 GC 스펙**:
    *   **PhD_Book_Author의 학술 스펙**: "자바스크립트 엔진에서 중첩 함수가 선언되면, 해당 함수는 자신의 [[Environment]] 내부 슬롯에 현재 실행 중인 실행 컨텍스트의 렉시컬 환경(Lexical Environment)을 저장합니다. 외부 함수가 실행 완료되어 콜 스택에서 제거되더라도, 내부 함수(클로저)가 전역이나 다른 살아있는 객체에 의해 참조되는 한, 가비지 컬렉터(GC)는 도달 가능성(Reachability) 분석에 의해 상위 렉시컬 환경의 활성 객체(Environment Record)들을 메모리에서 해제하지 않고 보존합니다. 이는 렉시컬 스코프 체인이 런타임에 동적으로 유지되는 내부 명세 동작 메커니즘입니다."

## 🧪 실습 미션
*   **미션 파일**: `mission.js`
*   **요구 사항**:
    1.  `createStore(reducer, initialState)`: Redux 스타일의 클로저 기반 상태 관리 저장소 구현
        - 외부에서 직접 접근할 수 없도록 `state`와 `listeners` 배열을 함수 내부에 은닉합니다.
        - `getState()`: 현재 상태를 반환합니다.
        - `dispatch(action)`: `reducer(state, action)`의 결과를 새로운 `state`로 업데이트하고, 등록된 모든 `listeners` 콜백을 순차 호출합니다.
        - `subscribe(listener)`: 리스너 함수를 등록하고, 실행 시 해당 리스너를 목록에서 제거하는 `unsubscribe` 함수를 반환합니다.

## 💡 힌트 및 트러블슈팅
*   `subscribe` 내에서 등록된 `listener`를 해제하기 위해 배열의 `filter` 등을 활용하여 새로운 리스너 배열을 할당하세요. 반환하는 `unsubscribe` 함수 역시 외부의 `listeners` 배열 변수를 공유하는 렉시컬 클로저가 됩니다.
