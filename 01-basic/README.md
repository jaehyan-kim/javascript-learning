# JavaScript [기초 트랙]

이 트랙은 자바스크립트를 처음 배우는 입문자가 필수적으로 익혀야 할 핵심 개념을 담은 기초 실습 코스입니다. 한국의 자바스크립트 최신 교재들의 공통 목차 7대 표준 핵심 키워드를 기준으로 각 챕터가 구성되어 있습니다.

## 📅 학습 핵심 로드맵

- **Chapter 01: 변수와 타입 (Variables & Types)**
- **Chapter 02: 제어문 (Control Flows)**
- **Chapter 03: 객체와 배열 (Objects & Arrays) [확장판]**
- **Chapter 04: 함수와 렉시컬 스코프 (Functions & Closures)**
- **Chapter 05: DOM 조작 (DOM Manipulation)**
- **Chapter 06: 이벤트 핸들링 (Event Handling)**
- **Chapter 07: 비동기 API (Asynchronous API)**
- **Chapter 08: 종합 프로젝트 (기초 마일스톤 통합)**

## 🚀 학습 가이드

1.  각 챕터 폴더(`01-basic/chapters/chapter-0x/`)로 이동하여 `README.md` 명세와 핵심 이론을 숙지합니다.
2.  `mission.js` 파일 내의 `TODO` 주석 지시에 맞춰 자바스크립트 코드를 단계별로 작성합니다.
3.  구현 완료 후 루트 경로에서 아래 테스트 스크립트를 가동하여 코드를 검증하십시오.
    ```bash
    npm run test:basic
    ```
